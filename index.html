Widget Preview

This preview allows you to see how the widget appears in different themes. Use the controls to toggle between dark and light modes, and reset the widget state.

How to Use SupanoticeWidget

The SupanoticeWidget component provides a floating notification bubble that shows product news and announcements.

<script type="module" src="https://cdn.example.com/supanotice-widget.js"></script>
<supanotice-widget widget-id="your-widget-id"></supanotice-widget>

Attributes

  • widget-id: Set a unique identifier for your widget to fetch its configuration (required)

Features

  • Fixed position bubble in the bottom-right corner of the page
  • Badge showing unread notification count
  • Responsive widget design
  • Support for tagging and marking notifications as read

Preview Component

Additionally, we provide a preview component that can be used for testing and demonstration purposes:

<supanotice-preview>
  <supanotice-widget widget-id="your-widget-id" preview-mode></supanotice-widget>
</supanotice-preview>

The preview component allows you to:

  • Toggle between light and dark themes
  • Reset widget states (clear read notifications)
  • View the widget in a contained environment

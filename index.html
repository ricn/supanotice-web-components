How to Use SupanoticeWidget

The SupanoticeWidget component provides a floating notification bubble that shows product news and announcements.

Production Usage

<script type="module" src="https://cdn.example.com/supanotice-widget.js"></script>
<supanotice-widget widget-id="your-widget-id"></supanotice-widget>

Local Development

<supanotice-widget 
  widget-id="your-widget-id" 
  api-endpoint="http://localhost:4000">
</supanotice-widget>

Attributes

  • widget-id: Set a unique identifier for your widget to fetch its configuration (required)
  • api-endpoint: Override the API endpoint URL (optional, defaults to https://supanotice.com)
  • preview-mode: Enable preview mode for use within containers (optional)

Open widget via link

Add data attributes to any clickable element to control the widget:

<a href="#" data-supanotice-open data-supanotice-target="#myWidget">Open announcements</a>
<a href="#" data-supanotice-toggle data-supanotice-target="#myWidget">Toggle</a>
<a href="#" data-supanotice-close data-supanotice-target="#myWidget">Close</a>

Targeting options:

  • data-supanotice-target: CSS selector for the widget element (e.g. #myWidget)
  • data-supanotice-widget: match a specific widget-id if you prefer id-based targeting

Programmatic API:

const el = document.querySelector('supanotice-widget');
el.open();   // or el.toggle(), el.close()

Features

  • Fixed position bubble in the bottom-right corner of the page
  • Badge showing unread notification count
  • Responsive widget design
  • Support for tagging and marking notifications as read

Preview Component

Additionally, we provide a preview component that can be used for testing and demonstration purposes:

<supanotice-preview>
  <supanotice-widget widget-id="your-widget-id" preview-mode></supanotice-widget>
</supanotice-preview>

The preview component allows you to:

  • Toggle between light and dark themes
  • Reset widget states (clear read notifications)
  • View the widget in a contained environment

SupanoticeWidget Component Demo

This page demonstrates the SupanoticeWidget component, which provides a floating notification bubble that shows product news and announcements.

How to Use

Simply add the following script to your HTML:

<script type="module" src="https://cdn.example.com/supanotice-widget.js"></script>
<supanotice-widget></supanotice-widget>

Features

  • Fixed position bubble in the bottom-right corner of the page
  • Badge showing unread notification count
  • Responsive widget design
  • Support for tagging and marking notifications as read
